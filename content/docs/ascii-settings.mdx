---
title: ASCII Settings
description: Configure the ASCII text overlay appearance
---

## Font Size

Controls the size of the monospace characters. Smaller sizes = more characters = more detail. The grid automatically recalculates when you change font size.

## Line Height

The vertical spacing between lines. Default is `0.78` (tight). Lower values pack lines closer together for a denser look.

## Letter Spacing

Horizontal spacing between characters. Default is `0.06em`.

## Color

The base color of the ASCII text. Works in combination with the blend mode.

## Opacity

How visible the ASCII layer is over the background image. Default is `0.38` for a subtle overlay.

## Blend Mode

How the ASCII text composites with the background image:
- **Screen** — additive blending, text lightens the image (default)
- **Normal** — opaque text on top of the image
- **Multiply** — darkening blend

## Character Ramp

The set of characters used to represent brightness levels. Darker areas of the image use denser characters (like `#`, `@`), lighter areas use sparser ones (like `.`, ` `).
