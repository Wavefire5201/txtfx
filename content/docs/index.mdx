## What is txtfx?

txtfx is a real-time ASCII art engine that composites animated character effects over photographs. Upload an image, add effects, and share your creation.

## Quick Start

1. **Open the editor** at [/editor](/editor)
2. **Upload an image** — click or drag an image into the canvas
3. **Add an effect** — click "Add Effect" in the right panel and choose one
4. **Adjust settings** — use the sliders to tweak colors, speed, glow, and more
5. **Share** — click the Share button to get a link

## Editor Layout

- **Left panel** — Tools (brush, pan) and layer visibility toggles
- **Center** — Canvas with your image and ASCII overlay
- **Right panel** — ASCII settings, effect controls, presets
- **Bottom** — Timeline with playback controls and effect bars

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `B` | Foreground brush |
| `N` | Background brush |
| `V` | Pan tool |
| `M` | Toggle mask |
| `[ ]` | Brush size |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
