## Embed Snippet

Export an embed snippet to include your ASCII art scene in any website:

1. Click **Export** in the toolbar
2. Choose **Copy Embed Snippet**
3. Paste the HTML into your page

The embed is a self-contained HTML snippet with inline CSS and JavaScript that recreates the ASCII effect animation.

## Standalone HTML

Export a full standalone HTML file that can be opened directly in any browser:

1. Click **Export** in the toolbar
2. Choose **Export HTML**
3. Open the downloaded `.html` file

The file includes the image, all effect settings, and the rendering engine.

<Callout type="info">
The standalone HTML export is being rebuilt to support all new features (multi-color effects, masks, etc.). Some effects may render differently in the export.
</Callout>
